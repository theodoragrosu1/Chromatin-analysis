This is a folder containing the scripts for pygenometracks and the deeptools heatmaps. it includes data from both 3.ATACseq and 4. Cut and Run repos. It also requires a .gtf file for hg38 gene annotation, as well as some public datasets

* the tracks file has the instructions for how every tracks was constructed
* pyegnometracks contain the loci selected and plotted
* Myb ChIPseq: GSM4420242 and GSM4420243
* Jurkat HiC: GSE206131
* GSM4851798: Jurkat H3K27me3 ChIPseq
* GSM4851799: Jurkat H3K27ac ChIPseq
